# Discovery Questionnaire

## Business Context

- Which business outcomes should AI modernization improve?
- Which workflows are most constrained by manual analysis or delayed information?
- Who owns the success metrics for each target use case?
- Which user groups will consume AI-enabled capabilities?

## Data and Integration

- Which source systems contain the highest-value operational data?
- Are CDC logs available for core transaction systems?
- Which datasets have known quality or ownership issues?
- What data classifications and privacy constraints apply?
- Which data products already exist?

## Platform and Operations

- Which cloud and Kubernetes platforms are approved?
- What CI/CD tools are standard?
- What observability platform is used?
- How are secrets, certificates, and identities managed?
- What release controls apply to data and AI services?

## AI Governance

- Which model providers are approved?
- What prompts and responses must be logged?
- What retrieval context should be auditable?
- What content filters and policy checks are required?
- What human review process applies to high-risk use cases?

## Adoption

- How will users provide feedback?
- What process changes are required?
- What training is needed for product, operations, and support teams?
