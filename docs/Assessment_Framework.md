# Assessment Framework

This public framework supports structured discovery for enterprise AI modernization. It does not include proprietary scoring logic, client-specific benchmarks, or commercial assessment models.

## Assessment Dimensions

| Dimension | Evaluation Focus |
| --- | --- |
| Business alignment | Use-case value, executive sponsorship, operating model |
| Data readiness | Data quality, ownership, lineage, cataloging, access controls |
| Integration maturity | CDC readiness, API coverage, event streaming patterns |
| AI platform maturity | Model gateway, vector search, evaluation, observability |
| Security and compliance | Identity, privacy, retention, auditability, risk review |
| Delivery capability | DevSecOps, platform engineering, reusable patterns |
| Change adoption | Training, workflow fit, feedback loops, support model |

## Workshop Evidence

- Current-state architecture diagrams
- Source system inventory
- Data product candidate list
- Security and compliance requirements
- AI use-case portfolio
- Current delivery pipeline
- Observability and incident data

## Maturity Levels

| Level | Description |
| --- | --- |
| 1 - Ad hoc | Pilots depend on manual data and isolated tools |
| 2 - Repeatable | Common patterns exist but are not platformized |
| 3 - Managed | Shared services and governance are consistently applied |
| 4 - Optimized | AI delivery is measured, reusable, and continuously improved |

## Deliverables

- Current-state summary
- Target-state architecture
- Gap analysis
- Use-case prioritization narrative
- Modernization roadmap
- Risk and dependency register

## Public Boundary

Keep scoring weights, proprietary calculators, internal benchmarks, and client-specific recommendations outside this public repository.
