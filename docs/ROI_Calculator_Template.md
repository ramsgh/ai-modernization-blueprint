# ROI Calculator Template

Use this public template to structure ROI discussions. Do not store client financials in this repository.

## Inputs

| Category | Example Input |
| --- | --- |
| Labor efficiency | Hours saved per workflow per month |
| Cycle-time reduction | Days reduced in process completion |
| Error reduction | Avoided rework or defect cost |
| Revenue enablement | Faster quote, fulfillment, or retention impact |
| Risk reduction | Avoided incidents, audit findings, or penalties |
| Platform reuse | Reduced duplicate infrastructure and delivery effort |

## Calculation Pattern

```text
Annual gross benefit =
  labor benefit
  + cycle-time benefit
  + error reduction benefit
  + revenue enablement benefit
  + risk reduction benefit
  + platform reuse benefit

Annual net benefit =
  annual gross benefit
  - annual run cost
  - annual change management cost
```

## Public Boundary

Keep proprietary assumptions, assessment scoring logic, client-specific financial data, and commercial models outside public GitHub repositories.
